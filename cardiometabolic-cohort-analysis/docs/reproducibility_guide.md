# Reproducibility Guide
1. **Environments.** Python >= 3.10 with `python/requirements.txt` (no survival-specific package required). R >= 4.2 with `r/requirements.R`. SAS 9.4 / SAS Studio for the SAS programs. 2. **Determinism.** Fixed seed (`SEED=20240607`); re-running reproduces the same raw file and outputs. 3. **Order.** `python/run_all.py` runs steps 01->11. R: `source("r/run_all.R")` (it creates output dirs first). SAS: set `%let root` in `sas/config.sas`, then submit `sas/run_all.sas` from the repo root. 4. **One data source.** All three languages read `data/raw/synthetic_cardiometabolic_raw.csv`. 5. **Validation.** `validation/validate_python_outputs.py` checks all 21 tables, 6 figures, model summaries, dashboard exports, raw/analysis-ready/patient counts, data-dictionary coverage, and headline metrics; `compare_language_outputs.py` aligns metrics across languages after R/SAS run.
