# Outcome Definitions
- **hba1c_12m** (%): continuous, linear-regression outcome.
- **bp_controlled_12m**: 1 if 12-month systolic BP < 140 mmHg, else 0; logistic-regression outcome.
- **mace_event / months_to_event**: MACE/death indicator and time from enrollment to event or administrative censoring; survival outcome.
- **Monthly MACE count**: number of events with `event_date` in each calendar month; the time-series unit.
