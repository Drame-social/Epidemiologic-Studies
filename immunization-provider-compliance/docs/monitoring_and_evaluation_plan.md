# Monitoring and evaluation plan

## Purpose

This M&E module extends the epidemiologic analysis into routine program monitoring and practical evaluation. The program question is no longer only, "Which characteristics are associated with non-compliance?" It also asks whether the compliance program is reaching high-risk providers, whether data quality is strong enough for quarterly decision-making, and whether performance is moving toward program targets.

## Monitoring questions

1. Are site-visit records complete enough for routine program monitoring?
2. Which compliance domains are above target thresholds?
3. Is technical assistance reaching high-risk providers and visits?
4. Are repeat non-compliant providers being observed again in a reasonable time window?
5. Are non-compliance and data-quality patterns different by provider type, rural/urban status, or jurisdiction?
6. Which indicators should be reviewed quarterly by program leadership?

## Evaluation questions

1. Did non-compliance change after the simulated 2022 technical-assistance targeting period?
2. Is prior technical assistance associated with lower adjusted odds of non-compliance after controlling for provider and visit characteristics?
3. Are high-risk providers receiving support at a rate consistent with program targets?
4. Are the observed improvements large enough to justify operational changes, or are they mostly small fluctuations in routine data?

## Data source

The M&E module uses the same synthetic provider compliance dataset as the rest of the project. The unit of analysis for most indicators is the provider site visit. Provider-level indicators aggregate visits to the provider level.

## Indicator review frequency

Quarterly review is used because it is realistic for public health program monitoring. Monthly data are available for time-series summaries, but quarterly reporting is more stable for program decisions and less likely to overreact to small-cell fluctuations.

## Main limitation

The dataset is synthetic and does not include a true randomized intervention, budget data, staffing counts, or a complete operational follow-up log. Follow-up indicators therefore use observed later site visits as a proxy, and evaluation findings should be interpreted as program-monitoring signals rather than causal estimates.
